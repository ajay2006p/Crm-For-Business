Integrations & APIs

These are the external services this app uses, what they do, and where. Edit the settings here directly.

WhatsApp (Click-to-chat)
No API key needed

What: Opens WhatsApp with a pre-filled message. Where: Outreach campaigns, Meetings (notify team), Tasks (assign/notify). Free, uses wa.me links.

OpenStreetMap Nominatim
No API key needed

What: Looks up cities & areas worldwide. Where: Area API tab and lead scraping by area. Public, free.

Google Maps Scraper (Playwright)
No API key needed

What: Collects business leads from Google Maps. Where: Leads → Scrape New Leads. Runs locally in a headless browser.

Twilio SMS (optional)
Optional

What: Sends automated SMS. Where: Outreach when channel = SMS. Needs a paid Twilio account. Leave blank to use WhatsApp instead.
